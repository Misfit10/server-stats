# Server Performance Stats Script

This project contains a Bash script that analyzes basic Linux server performance statistics.  
It is designed to run on any standard Linux server without installing additional tools.

## Features
- Displays total CPU usage
- Shows memory usage (used vs free with percentage)
- Shows disk usage (used vs free with percentage)
- Lists top 5 processes by CPU usage
- Lists top 5 processes by memory usage

## Requirements
- Linux OS
- Bash shell
- Standard Linux utilities (top, free, df, ps)

## How to Run
Clone the repository and run the following commands:

```bash
chmod +x server-stats.sh
./server-stats.sh
